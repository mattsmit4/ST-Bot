What is ST-Bot?

ST-Bot is a conversational AI assistant built to help customers navigate StarTech.com's product catalog of 5,162 technology products. Instead of browsing filters and spec sheets, customers describe what they need in plain English and the bot finds the right products.

Built with Streamlit, powered by gpt-5.4-nano for understanding queries, extracting product specs, and generating responses. The search engine uses a two-phase scoring system — hard requirements eliminate wrong products instantly, then soft filters rank remaining matches by relevance.

Architecture (v2.0)

The pipeline follows a 6-layer flow:

  • 1. Storefront — Chat UI, the only layer the customer sees
  • 2. Sorter — Sanitizes input, loads conversation state, prevents manipulation
  • 3. Intent Classifier — AI identifies the type of question (8 intents, 2 paths)
  • 4. Filter Extractor — AI pulls specs from the query into structured search criteria
  • 5. Handlers — One handler per intent, each with specialized logic
  • 6. Response & Side Effects — Each handler generates its own response; system updates conversation state

Testing

Three layers of automated testing protect against regressions:

  • 62 unit tests — Fast tests for search scoring, connector matching, and field normalization
  • 22 integration tests — Full pipeline tests with simulated AI responses
  • End-to-end test harness — Real AI calls and real product data through the full pipeline
